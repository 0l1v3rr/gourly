# gourly
✂️ This is a simple URL shortener.

<hr>

## Tech stack
- **Frontend:** React, Tailwind, Axios
- **Backend:** Go, Gin, Gorm
- **Database:** MySQL 

<hr>

## Build from source
Clone this project:
```sh
git clone https://github.com/0l1v3rr/gourly.git
cd gourly
```

### Run the frontend:
```sh
cd view

# if you're running the app for the first time
npm i

npm star
```

### Running the bakcend:
```
cd app
go run main.go
```
